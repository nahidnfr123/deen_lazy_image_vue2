# deen_lazy_image_vue2

## Plugin Installation

Yarn:

```
yarn add deen_lazy_image_vue2
```

###

Npm:

```
npm install deen_lazy_image_vue2
```

Usage

```
<deen-lazy-image
    source="https://www.pixelstalk.net/wp-content/uploads/2016/07/4k-Wallpapers-Screen-Download.jpg"
    height="auto"
    width="100%"
    imageClass=""
    alt="Image"
    imgStyle=""
    />
```
